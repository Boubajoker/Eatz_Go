# Eatz_Go :hamburger:
An IA that can prupose you what you can eat !!

# How to use it ?

Just execute <a href="main.py">'main.py'</a> for devs.
And execute <a href="EatzGo.exe">EatzGo.exe</a> for normal using.

And that's it, the IA will prupose to you a meal !
